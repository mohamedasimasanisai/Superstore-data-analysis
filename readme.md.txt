## Overview
This project analyzes historical sales data from a global superstore to extract business insights using Python.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Key Questions Answered
- Which categories generate the most revenue?
- How do monthly sales change over time?
- What is the impact of discounts on profit?

## Summary of Insights
- Technology has the highest total sales.
- Furniture often leads to negative profit due to high discounts.
- Sales peak in November and December.

## How to Run
Clone the repo and run `project_for_data_analysis.py`. Make sure you have the libraries in `requirements.txt` installed.
